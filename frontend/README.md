# SparePartFinder Frontend

React.js frontend for SparePartFinder Pro.

## Setup
```bash
npm install
npm run dev
```

Runs on: http://localhost:5173

## Features
- Dashboard with analytics
- AI image analyzer
- Scan history (localStorage)
- Dark/Light theme
- Price comparison display

## Tech Stack
- React 18
- Vite
- React Router
- Axios
- Chart.js

## Build for Production
```bash
npm run build
```
